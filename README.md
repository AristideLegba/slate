# slate

A free online foundation for schools.

## Developing

### On Windows

- Install choco, have windows 10 pro
- `choco install docker`
- `choco install habitat`
- `choco install git`
- Clone and `cd` into slate repository
- `$env:HAB_DOCKER_OPTS="-p 7080:7080"`
- `hab studio -D enter`
